Open-Drain I2C SDA line
Design i2c_line to model an open-drain I²C SDA line using tri1.

Module: i2c_line
Input: drive_low (1 = actively pull line low, 0 = release)
Output: sda (idle high when undriven)
code:
module i2c_line(
    input drive_low,
    output tri1 sda
);
assign sda = ~(drive_low);
endmodule
