Shared Line using wor
Two alarm sources share a siren line. If either asserts, siren must be 1.

Module: alarm_siren
Inputs: alarm1, alarm2
Output: siren (use wor so multiple drivers OR together)
code:
module alarm_siren(
    input alarm1,
    input alarm2,
    output wor siren
);
assign siren = alarm1 | alarm2;
endmodule
